1. [Adalo](https://www.adalo.com/) – Build mobile and web apps with drag-and-drop; beginner-friendly.
2. [Bubble](https://bubble.io/) – Powerful visual web app creator with workflows and database support.
3. [Webflow](https://webflow.com/) – Visual website builder that outputs clean code and scalable hosting.
4. [Glide](https://www.glideapps.com/) – Turn spreadsheets into polished apps quickly.
5. [Softr](https://www.softr.io/) – Create web apps from Airtable or Google Sheets without code.
6. [Thunkable](https://thunkable.com/) – No-code platform for native mobile apps on iOS and Android.
7. [AppSheet](https://www.appsheet.com/) – Google-backed tool to build apps from data sources like Sheets.
8. [Bildr](https://www.bildr.com/) – Flexibly build web apps and sites visually.
9. [WeWeb](https://www.weweb.io/) – Visual builder for dynamic web applications.
10. [Momen](https://momen.co/) – Easy no-code tool for apps with AI features.
11. [Carrd](https://carrd.co/) – Super-simple one-page site builder, often free.
12. [Typeform](https://www.typeform.com/) – Create interactive forms and surveys with no code.
13. [Hostinger Website Builder](https://www.hostinger.com/website-builder) – Affordable, beginner-friendly web creation tool.
14. [Claude / Anthropic tools](https://www.anthropic.com/) – Build interactive AI apps through conversational prompts.
15. [GitHub Spark](https://github.com/) – Tools from GitHub to help build full-stack apps with low-code helpers.
16. [Firebase](https://firebase.google.com/) – Free tier backend with auth, database, and hosting tools.
17. [Supabase](https://supabase.com/) – Open-source backend as a service with realtime Postgres.
18. [Backendless](https://backendless.com/) – Visual backend builder with free tier options.
19. [Hasura](https://hasura.io/) – Instant GraphQL over your database, minimal code needed.
20. [Xano](https://xano.com/) – No-code scalable backend with automation logic.
21. [Zapier](https://zapier.com/) – Connect apps and automate tasks without code.
22. [Make (Integromat)](https://www.make.com/) – Visual automation builder with robust logic.
23. [IFTTT](https://ifttt.com/) – Simple triggers connecting apps and services.
24. [Parabola](https://parabola.io/) – Automate and transform data without coding.
25. [n8n](https://n8n.io/) – Open-source workflow automation you can self-host.
26. [Retool](https://retool.com/) – Build internal dashboards with drag-and-drop and optional code.
27. [Appsmith](https://www.appsmith.com/) – Open-source low-code tool for internal apps and dashboards.
28. [Budibase](https://budibase.com/) – Build internal business apps quickly with automation.
29. [Internal.io](https://internal.io/) – No-code builder focused on internal tooling (free tier available).
30. [Microsoft Power Apps](https://powerapps.microsoft.com/) – Low-code platform tied into Microsoft 365.
31. [Zoho Creator](https://www.zoho.com/creator/) – Build custom business apps with low-coding.
32. [Kissflow](https://kissflow.com/) – Low-code tool for business process apps and automations.
33. [Quickbase](https://www.quickbase.com/) – Build custom workflow apps with minimal code.
34. [OutSystems](https://www.outsystems.com/) – Enterprise low-code for complex apps (free trial).
35. [Appian](https://www.appian.com/) – Low-code process automation and case management.
36. [Figma Make (AI Prototyping)](https://www.figma.com/) – Turn designs into working app prototypes with AI.
37. [Thunkable AI](https://thunkable.com/) – AI-assisted mobile app builder.
38. [Automate.io](https://automate.io/) – AI workflow builder tied to cloud apps (free tier).
39. [Voiceflow](https://www.voiceflow.com/) – Build voice and chat experiences without code.
40. [Notion Automations](https://www.notion.so/) – Built-in no-code logic for Notion databases and pages.
